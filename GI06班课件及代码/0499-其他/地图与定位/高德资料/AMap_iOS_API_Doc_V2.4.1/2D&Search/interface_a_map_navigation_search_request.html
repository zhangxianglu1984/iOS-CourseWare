AMapNavigationSearchRequest类 参考

路径规划查询参数类。 更多...

#import <AMapSearchObj.h>

类 AMapNavigationSearchRequest 继承关系图:

属性

AMapSearchType searchType
 查询服务类型，可选项，具体参见 AMapSearchType 定义。
 
NSInteger multipath
 是否提供备选步行方案（默认为0），0：只提供一条步行方案；1：提供备选步行方案（有可能无备选方案），步行路径规划时填写。
 
NSArray * waypoints
 途经点数组，数组中存放 AMapGeoPoint 对象，驾车路径规划时填写。
 
NSArray * avoidpolygons
 避让区域数组，数组中存放 AMapGeoPolygon 对象，支持32个避让区域，每个区域最多可有16个顶点，驾车路径规划时填写。
 
NSString * avoidroad
 避让道路名，只支持一条避让道路，驾车路径规划时填写。
 
NSString * originId
 出发点 POI ID，当起点为POI中获取坐标时建议填充此值，驾车路径规划时填写。
 
NSString * destinationId
 目的地 POI ID，当起点为POI中获取坐标时建议填充此值，驾车路径规划时填写。
 
BOOL nightflag
 是否包含夜班车，YES为包含夜间线路，NO为不包含夜间线路， 公交路径规划时填写。
 
NSInteger strategy
 计算策略（默认为0）。
驾车导航策略：0-速度优先（时间）；1-费用优先（不走收费路段的最快道路）；2-距离优先；3-不走快速路；4-结合实时交通（躲避拥堵）；5-多策略（同时使用速度优先、费用优先、距离优先三个策略）；6-不走高速；7-不走高速且避免收费；8-躲避收费和拥堵；9-不走高速且躲避收费和拥堵。
公交换乘策略：0-最快捷模式；1-最经济模式；2-最少换乘模式；3-最少步行模式；4-最舒适模式；5-不乘地铁模式。

NSString * city
 城市。
 
BOOL requireExtension
 是否返回详细信息（默认为NO），驾车路径规划时填写。
